# DeFi Use Cases

## Summary
Orbs의 대표 유즈케이스는 DeFi 안에서의 고급 주문 실행과 거래 인프라 제공입니다. 공개 자료상 dTWAP, dLIMIT, dSLTP, Liquidity Hub, Perpetual Hub는 모두 DeFi 사용자 경험을 CeFi 수준으로 끌어올리는 데 초점을 둡니다.

## Main DeFi Primitives
- dTWAP: decentralized time-weighted average price orders
- dLIMIT: decentralized limit orders with on-chain execution guarantees
- dSLTP: decentralized stop-loss and take-profit orders with on-chain execution guarantees
- Liquidity Hub: liquidity optimization / aggregation for DEXs
- Perpetual Hub: intent-based perpetual futures infrastructure

## Working Interpretation
Orbs는 DeFi 프로토콜이 native smart contract만으로 구현하기 어려운 주문 유형과 execution logic을 탈중앙화된 백엔드 형태로 제공하는 것으로 보입니다. 따라서 DeFi use case 문맥에서 Orbs의 가치는 "새로운 금융 상품 자체"보다 "기존 DeFi 거래 경험을 고도화하는 실행 인프라"에 있습니다.
