# Use Cases Overview

## Summary
Orbs의 현재 유즈케이스는 매우 분명하게 DeFi 실행 인프라 쪽에 집중되어 있습니다. 공개 자료 기준 핵심 축은 advanced trading orders, aggregated liquidity, decentralized derivatives, 그리고 autonomous DeFi agents / execution services 입니다.

## Main Categories
- Advanced trading orders
  - dTWAP
  - dLIMIT
  - dSLTP
- Liquidity optimization
  - Liquidity Hub
- Decentralized derivatives / perpetuals
  - Perpetual Hub
- Agent execution / AI-oriented DeFi automation
  - Orbs Agentic
- General-purpose decentralized execution services
  - Orbs VM
  - Orbs Lambda

## Working Interpretation
현재의 Orbs는 범용 체인 유즈케이스보다, DeFi 프로토콜이 자체적으로 구현하기 어려운 복잡한 주문 로직과 실행 로직을 외부 분산 실행 계층으로 제공하는 방향이 강합니다. 즉, Orbs는 사용자-facing app이라기보다 DEX / DeFi protocol을 위한 backend or middleware 성격이 큽니다.

## Key Themes
- CeFi-level execution to DeFi
- Decentralized algorithmic orders
- Better liquidity sourcing / aggregation
- Intent-based derivatives / perpetuals
- Execution layer for autonomous agents
