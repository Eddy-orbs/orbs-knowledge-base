## User-Provided Current Reward Parameters
- 2026-04-29 user update: current delegator reward rate is 6.67% annualized average.
- 2026-04-29 user update: rewards accrue continuously at the contract level, effectively every block, as a numerical reward balance. They are not automatically added to the user's actual staked ORBS amount. The user must manually claim / settle rewards in TETRA or another official staking interface; only that claim / settlement action adds the accrued rewards to the staked amount and makes the reward portion part of the unlockable / withdrawable staking balance.
- 2026-04-29 user update: moving or transferring rewarded/staked ORBS requires unlocking / unstaking.
- 2026-04-29 user update: Guardian reward rates are 10% on the Guardian's own stake and 3.33% on delegated stake.

Use these as dated operational notes and recheck live official staking analytics or contract state when precision is required.

## Custodial Staking Reference
- Bithumb exchange: http://bithumb.co.kr/
- 2026-04-29 user update: Bithumb provides a custodial ORBS staking service in Korea.
- 2026-04-29 user update: the service allows users to deposit ORBS on Bithumb and opt into staking with a simple consent flow.
- 2026-04-29 user update: exchange-side advantage is no lockup period for trading inside Bithumb, but reward rate is lower than on-chain staking, generally around 2–4%.

Use this as a dated custodial-staking note. Current terms, reward rates, fees, eligibility, and risks should be checked directly on Bithumb.
