# Exchange Listings

## Summary
ORBS exchange availability can be checked through the markets tabs on major market-data aggregators. Because this information changes frequently, it is safer to document the reference sources and checking method than to maintain a fixed static exchange list.

## Primary Market References
- CoinMarketCap Markets: https://coinmarketcap.com/currencies/orbs/#Markets
- CoinGecko Markets: https://www.coingecko.com/en/coins/orbs

## Recommended Documentation Policy
- Do not maintain a detailed fixed exchange table in static documentation
- Treat CoinMarketCap and CoinGecko as the default live references
- If a dated snapshot is ever needed, create it only for that moment and do not treat it as permanent canonical knowledge

## Notes
- The current approach is to maintain the source map and check live market data when needed
- Market-data pages are better treated as live references than as permanent static facts
- If a user asks for the latest listing status, direct them first to the CoinMarketCap or CoinGecko market pages
