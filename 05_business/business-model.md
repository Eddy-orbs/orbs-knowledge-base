# Business Model

## Summary
현재 공개 자료 기준으로 Orbs의 비즈니스 모델은 단순한 체인 자체 판매보다, DeFi 프로토콜과 DEX가 활용할 수 있는 execution infrastructure / backend layer 제공에 더 가깝습니다. 즉, Orbs는 고급 주문 로직, 유동성 최적화, 파생상품 실행, agent execution 같은 기능을 제품군으로 제공하는 방향으로 보입니다.

## Core Logic
- DeFi protocol / DEX integration driven model
- Layer-3 execution services as infrastructure
- Product suite centered around dLIMIT, dTWAP, dSLTP, Liquidity Hub, Perpetual Hub, and Agentic
- Network secured and economically coordinated by ORBS token incentives

## Business Interpretation
Orbs의 현재 사업 방향은 "토큰만 있는 체인"보다는, 실제로 통합 가능한 execution products를 제공하고 그 adoption을 통해 생태계 영향력을 넓히는 구조에 가깝습니다. 따라서 business narrative는 기술 자체보다도, 어떤 DEX/DeFi 앱이 Orbs product를 붙여서 거래 경험을 개선하는가에 의해 설명되는 편이 더 자연스럽습니다.

## Key Revenue / Value Hypotheses To Track
- protocol integrations as growth engine
- value accrual through network usage and token-aligned incentive layer
- ecosystem expansion through products becoming a de facto execution standard

## Open Questions
- 공식적으로 수익모델을 어떻게 서술하는가?
- product usage가 ORBS token value accrual과 어떤 관계를 갖는가?
- business development의 핵심 KPI는 integrations, volume, staking participation 중 무엇인가?
