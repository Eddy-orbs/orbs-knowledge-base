# ORBS Token Overview

## Summary
ORBS는 Orbs 프로토콜의 핵심 네트워크 토큰입니다. 현재까지 정리된 자료 기준으로 ORBS는 실행 관련 수수료 정산, staking, Guardian 선출, PoS 참여자 보상에 연결되며, 단순 거래용 자산이 아니라 프로토콜 인센티브 레이어의 중심 역할을 합니다.

## Core Role
- fee settlement related to app / infrastructure execution
- staking asset
- Guardian election / network security participation
- reward distribution for PoS participants

## Cross-Chain Presence
ORBS는 여러 네트워크에서 참조되는 토큰 컨트랙트 주소가 존재하며, bridge를 통해 이동 가능한 구조가 정리되어 있습니다. 다만 canonical token 기준과 각 체인별 운용 방식을 공식 문서 기준으로 추가 확인하는 것이 좋습니다.
