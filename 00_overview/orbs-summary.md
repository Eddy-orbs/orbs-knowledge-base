# Orbs Summary

## One-line Summary
Orbs is a blockchain infrastructure project currently positioned as a decentralized Layer-3 and execution layer focused on bringing CeFi-level execution capabilities to DeFi.

## What This File Answers
- Orbs는 어떤 프로젝트인가?
- 지금 Orbs는 자신을 어떻게 포지셔닝하는가?
- 기술/비즈니스 관점에서 핵심 포인트는 무엇인가?

## Key Facts
- Official messaging emphasizes advanced on-chain trading and DeFi execution.
- Orbs uses a Proof-of-Stake model.
- Public materials connect Orbs staking and token infrastructure to Ethereum, with later expansion toward Polygon-based staking.
- Product/use-case pages highlight dLIMIT, dTWAP, Liquidity Hub, Perpetual Hub, Agentic, and broader execution services.

## Short Overview
현재 기준으로 Orbs는 범용 L1을 지향한다기보다, DeFi 프로토콜과 DEX에 붙는 실행 인프라 계층으로 읽힙니다. 공식 문구상 핵심 메시지는 "CeFi-level execution to DeFi"이며, 고급 주문 유형, 유동성 집계, 파생상품 지원, execution services 같은 기능에 초점을 두고 있습니다.

기술적으로는 PoS를 기반으로 하며, 공개 자료상 ORBS 토큰과 staking 계약은 Ethereum과 연결되어 있고, 이후 Polygon 쪽 확장도 진행된 것으로 보입니다. 다만 초기 포지셔닝과 현재 포지셔닝 간 변화는 추가 문서 검증이 필요합니다.
