# Prompt Examples

## Example Questions
- Orbs의 설립 역사 요약해줘
- Orbs의 현재 비즈니스 모델 설명해줘
- Orbs의 PoS 구조를 쉽게 설명해줘
- Orbs가 왜 Layer-3로 불리는지 설명해줘
- Orbs의 dLIMIT와 dTWAP 차이를 설명해줘
- Orbs의 파트너 구조를 제품별로 분류해줘
- Orbs의 virtual chains 개념을 현재 execution layer 포지셔닝과 연결해서 설명해줘
- Orbs VM과 Lambda의 차이를 설명해줘
- Orbs를 일반적인 L1 블록체인과 비교해줘

## Example Answer Style Notes
- 먼저 짧은 결론을 준다
- 역사 / 기술 / 사업 관점을 섞지 말고 구분한다
- 현재 포지셔닝과 초기 구조가 다르면 둘을 구분해서 설명한다
- 불확실한 founder biography, tokenomics 수치, 파트너 상태는 단정하지 않는다
