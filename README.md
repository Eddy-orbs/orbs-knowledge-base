# Orbs Knowledge Base

이 폴더는 Orbs 프로젝트 전용 지식베이스입니다.

## 목적
- Orbs 관련 사실 조회
- 역사, 기술, 비즈니스, 파트너십 맥락 설명
- Q&A 및 브리핑용 자료 축적

## 운영 원칙
- 공식 자료 우선
- 사실, 해석, 미확인 정보를 구분
- 긴 원문은 `07_sources/`에, 정제된 학습용 문서는 카테고리 폴더에 저장
- 최신 변경사항은 `08_notes/update-log.md`에 기록

## 추천 작업 순서
1. `07_sources/raw-links.md`에 원자료 링크 적재
2. `07_sources/official/` 아래에서 출처를 주제별로 정리
3. `00_overview/key-facts.md` 작성
4. `00_overview/timeline.md` 작성
5. founders / history / technology / use-cases / business 문서 채우기
6. `09_training/glossary.md`, `09_training/entity-map.md` 정리
7. `06_qna/faq.md`와 예상 질문 문서 작성

## 현재 상태
- 1차 knowledge base 골격 완성
- overview / history / founders / technology / use-cases / business / partners / faq / glossary 작성됨
- 일부 파트너 데이터와 founder bio는 추가 교차검증 시 더 단단해질 수 있음
